# ⚖️ Estudio Jurídico — Dra. Rocío D'Agostino

Sitio web profesional desarrollado para el estudio jurídico de la Dra. Rocío D'Agostino, ubicado en Luján, Buenos Aires.

El proyecto fue diseñado con el objetivo de transmitir confianza, claridad y profesionalismo, ofreciendo una presencia digital moderna orientada a la captación de nuevos clientes.

---

## 🌐 Sitio web

🔗 https://legal-rocio.netlify.app/

---

## 🧩 Descripción

Este sitio funciona como carta de presentación digital del estudio, permitiendo a los usuarios:

* Conocer los servicios legales ofrecidos
* Acceder a información clara y accesible
* Establecer contacto directo de forma rápida
* Generar confianza a través de un diseño profesional

El enfoque estuvo puesto en la simplicidad, la legibilidad y la experiencia de usuario.

---

## 🎨 Características principales

* Diseño moderno y minimalista
* Interfaz responsive (adaptada a dispositivos móviles)
* Navegación clara e intuitiva
* Optimización para performance y carga rápida
* Identidad visual alineada al rubro legal

---

## 🛠️ Tecnologías utilizadas

* React
* Vite
* TypeScript
* CSS moderno

---

## 🎯 Objetivo del desarrollo

El proyecto fue desarrollado con fines comerciales reales, priorizando:

* Conversión de visitantes en potenciales clientes
* Posicionamiento profesional del estudio
* Escalabilidad para futuras mejoras (SEO, dominio propio, etc.)
