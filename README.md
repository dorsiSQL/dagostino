# ⚖️ Estudio Jurídico — Dra. Rocío D'Agostino

Sitio web profesional desarrollado para el estudio jurídico de la Dra. Rocío D'Agostino, orientado a brindar información clara, presencia digital y captación de clientes en Luján, Buenos Aires.

---

## 🚀 Demo

🔗 https://legal-rocio.netlify.app/

---

## 🧩 Tecnologías utilizadas

* React
* Vite
* TypeScript
* CSS moderno (responsive design)
* Deploy en Netlify (con opción de migración a Vercel)

---

## 📁 Estructura del proyecto

```
/
├── public/          # Archivos estáticos (favicon, imágenes, etc.)
├── src/             # Código fuente principal
│   ├── components/  # Componentes reutilizables
│   ├── pages/       # Vistas principales
│   └── assets/      # Recursos internos
├── index.html       # Entry point HTML
├── vite.config.ts   # Configuración de Vite
└── package.json
```

---

## ⚙️ Instalación y uso

Clonar el repositorio:

```bash
git clone https://github.com/dorsiSQL/dagostino.git
cd dagostino
```

Instalar dependencias:

```bash
npm install
```

Modo desarrollo:

```bash
npm run dev
```

Build de producción:

```bash
npm run build
```

Preview local del build:

```bash
npm run preview
```

---

## 🌐 Deploy

Actualmente deployado en Netlify.

Configuración:

```bash
Build command: npm run build
Publish directory: dist
```

Opcional: migración simple a Vercel.
